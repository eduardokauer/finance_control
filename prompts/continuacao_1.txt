CONTEXTO DO PROJETO
Objetivo:
Criar uma base de dados financeira pessoal com backend em FastAPI, banco no Supabase, deploy no Render, automação no Make, ingestão de arquivos via Google Forms/Google Drive e envio de análise financeira por email.

STACK / SERVIÇOS
- Backend: Python 3.11+, FastAPI
- Banco: Supabase usado apenas como Postgres
- Deploy: Render
- Orquestração: Make
- Entrada de arquivo: Google Forms + Google Drive
- Testes: pytest
- Desenvolvimento local: Docker Desktop no Windows
- API protegida por token fixo Bearer
