# Prompts

Esta pasta não é a fonte canônica do processo do projeto.

Prompts canônicos de processo ficam em `docs/`.

Referências oficiais atuais:
- `docs/pm_cycle_start_prompt.md`: início de ciclo PM/LLM
- `docs/project_context.md`: estado atual do projeto e do ciclo
- `docs/pm_workflow.md`: processo do PM/LLM
- `docs/codex_workflow.md`: processo do executor técnico

Arquivos mantidos em `prompts/` devem existir apenas por compatibilidade explícita ou utilidade específica, sem competir com `docs/` como fonte de verdade do workflow.
