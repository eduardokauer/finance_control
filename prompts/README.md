# Prompts

Esta pasta existe por compatibilidade e utilidade local.

O processo oficial do projeto vive fora do repositório.
Use:
- ChatGPT Project para refinamento, análise e geração de prompts
- Notion para registrar de forma curta a decisão e a fatia atual
- o repositório para contexto técnico estável e regras de execução

Arquivos em `prompts/` não devem competir com esses papéis.
